## Installation
Just download it and run it. Don't forget to set the right cluster using kubectx or whatever tool you prefer.

### Download
```
curl -Lo podloxx \
https://github.com/mogenius/podlox/releases/download/s/podloxx-1.0.2-darwin-arm64 \
&& chmod 755 podloxx

podloxx start
```
